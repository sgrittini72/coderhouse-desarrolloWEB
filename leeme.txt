Este Sitio utiliza SASS, animate.css.
Para poder utilizar estos módulos es necesario:
1- Intalar node.js dese https://nodejs.org/en/
2- Desde la carpeta del proyecto y de la terminal ejectuar:
   npm install
Esto va a generar la carpeta node-modules con los módulos necesarios.

Para procesar los SCSS desde la terminal y en la carpeta del proyecto, ejecutar:
1- npm run build-css
2- npm run watch-css
